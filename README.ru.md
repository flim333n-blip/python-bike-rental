# Аренда велосипедов

Простая консольная программа на Python, которая имитирует работу пункта проката велосипедов.

## Причина создания
Желание научиться программировать на Python

## Описание

Эта программа помогает автоматизировать процесс аренды велосипеда. Она последовательно:
1.  Проверяет, есть ли свободные велосипеды.
2.  Запрашивает возраст пользователя (аренда только с 18 лет).
3.  Для возрастных клиентов (59+ лет) даёт рекомендацию взять прогулку с гидом.
4.  Рассчитывает стоимость аренды (500 рублей в час).
5.  Предлагает выбрать способ оплаты: картой или наличными.

## Технологии

*   **Язык:** Python 3.13.4

## Как запустить

1.  Убедись, что у тебя установлен Python (версия 3.13.4 или выше).
2.  Скачай файл с кодом.
3.  Открой терминал (командную строку) в папке с файлом.
4.  Запусти программу командой:
    ```bash
    python bike_rental.py
    ```
